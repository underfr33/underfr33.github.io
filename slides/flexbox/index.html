Was ist Flexbox

  • Flexible Box Layout Module
  • Elemente einer Seite so anordnen, dass sie sich auf verschiedenen Bildschirmgrößen/Ausgabegeräten vorhersagbar verhalten
  • ersetzt float, table und inline-block
  • Flexbox-Modell besteht aus einem Eltern-Element flex-container, welches die Eigenschaften für die Anordnung der darin enthaltenen Kind-Elemente flex-items besitzt
  • Anordnung der Elemente lässt sich mit CSS steuern
+ flex container + flex items

Vorteile?

  • im Gegensatz zu block und inline ist Flexbox Richtungsunabhängig
  • keine festen Größen, abs. Positionierungen, JS/jQuery, float- oder CSS-Hacks
  • gleiche Höhe, vertikal zentrierter Content, Sticky Footers, column layouts
  • Größe der Kind Element ändert sich dynamisch
+ großer Vorteil für RWD + block: vertikal basiert + inline: horizontal basiert + Breite und Höhe der Kind Elemente ändern sich, um den verfügbaren Bildschirmbereich auf den verschiedenen Auflösungen (Devices) am besten zu füllen
